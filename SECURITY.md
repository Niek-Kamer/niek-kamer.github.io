# Security

This repository is a personal portfolio site. There is no auth, backend, or user data to compromise. That said, if you find:

- A way to inject or execute arbitrary content on the deployed site
- A leak of credentials, tokens, or anything that shouldn't be public
- A supply-chain risk in the build pipeline

please report it privately rather than opening a public issue.

**Contact:** open a [private security advisory](../../security/advisories/new) on this repository, or reach out via the contact on https://zkmcu.dev/.

Expect a response within a few hours. Thanks for taking the time.
